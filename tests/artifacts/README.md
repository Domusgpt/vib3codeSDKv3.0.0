# Test artifacts

This folder stores raw outputs for Phase 5 validation runs.

Subfolders:
- `telemetry/`: CLI telemetry exports (JSON/NDJSON)
- `exports/`: export hash manifests and golden snapshots
- `mcp/`: MCP tool call traces and responses
- `xr/`: XR benchmark logs and captures

Use `YYYYMMDD_HHMM_<scenario>` naming for artifacts.
